# mms-unpack

- Simple Script that Ceates and Unpack Boxes.
- Exmaple Usage 50 Tomatos + Empty Box = Tomato Box.
- 1 Tomato Box = Unpack in 50 Tomatos and a Empty Box.

# Features
 
- Pack / Unpack Boxes
